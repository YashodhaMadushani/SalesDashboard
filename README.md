# Sales Dashboard 2025
This repository contains a **Power BI dashboard** for 2025 sales data.

## Features
- Visualize sales by **state** and **product category**
- Filter by **Month Group** (Jan–Mar, Apr–Jun, Jul–Sep, Oct–Dec)
- Interactive charts for **profit, sales, and payment mode**

## Files
- `E-Commerce Sales Dashboard.pbix` – The Power BI Desktop file

## How to Use
1. Download the PBIX file
2. Open it in **Power BI Desktop**
3. Explore the dashboard interactively
